# Project Roadmap

## Phase 1 — Project Foundation & Architecture Setup

React/Vite frontend, layered FastAPI service, PostgreSQL schema, containerized local environment, health monitoring, tests, and architecture documentation.

## Phase 2 — SOC Dashboard

Authentication, analyst workflows, alert triage, interactive threat metrics, role-based access, and audit trails.

## Phase 3 — Threat Intelligence Engine

STIX/TAXII feeds, indicator normalization, enrichment, confidence scoring, deduplication, provenance, and correlation.

## Phase 4 — AI Threat Detection

Feature pipelines, governed model lifecycle, explainable detection, drift monitoring, human review, and adversarial evaluation.

## Phase 5 — Attack Path Prediction

Asset and identity graph, exposure modeling, probabilistic path ranking, remediation simulation, and validated analyst feedback loops.
