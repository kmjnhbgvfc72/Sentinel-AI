# AI Cyber Threat Intelligence — Landing Page

Standalone React, Vite, and TypeScript landing page for the completed AI Cyber Threat Intelligence System. It is isolated from the existing application and uses CSS-only visual assets plus Lucide icons.

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:5174`.

## Links

Dashboard, API documentation, and repository URLs are centralized in `src/config/links.ts`.
